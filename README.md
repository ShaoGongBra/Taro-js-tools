# Taro js-tools
js工具库，支持日期函数、gps转换函数、表单验证函数、导航函数、字符串处理函数、全局变量、url参数、md5加密、base64加解密

# 使用方法
> npm install taro-js-tools    
> or    
> yarn add taro-js-tools
